setFioDomainVisibility

fun setFioDomainVisibility(fioDomain: String, visibility: FioDomainVisiblity, maxFee: BigInteger, technologyPartnerId: String = ""): PushTransactionResponse

By default all FIO Domains are non-public, meaning only the owner can register FIO Addresses on that domain. Setting them to public allows anyone to register a FIO Address on that domain.

Parameters

fioDomain - FIO Domain to make public or private. Default is private.

visibility - FioDomainVisiblity

maxFee - Maximum amount of SUFs the user is willing to pay for fee. Should be preceded by getFee for correct value.

technologyPartnerId - (optional) FIO Address of the wallet which generates this transaction.
