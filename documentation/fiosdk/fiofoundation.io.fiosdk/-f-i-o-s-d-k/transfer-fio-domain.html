transferFioDomain

fun transferFioDomain(fioDomain: String, newOwnerFioPublicKey: String, maxFee: BigInteger, technologyPartnerId: String): PushTransactionResponse

Transfers a FIO Domain to a new owner.

Parameters

fioDomain - FIO Domain. Please note that FIO Domain is case insensitive. If upper case characters are passed in, they will be converted to lower case.

newOwnerFioPublicKey - FIO Public Key. FIO Public Key of the new owner. If account for that key does not exist, it will be created as part of this call.

maxFee - Maximum amount of SUFs the user is willing to pay for fee. Should be preceded by /get_fee for correct value.

technologyPartnerId - FIO Address of the wallet which generates this transaction.

Exceptions

FIOError -
