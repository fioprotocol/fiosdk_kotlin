recordObtData

fun recordObtData(fioRequestId: BigInteger, payerFioAddress: String, payeeFioAddress: String, payerTokenPublicAddress: String, payeeTokenPublicAddress: String, amount: Double, chainCode: String, tokenCode: String, status: String = "sent_to_blockchain", obtId: String, maxFee: BigInteger, technologyPartnerId: String = ""): PushTransactionResponse

Records information on the FIO blockchain about a transaction that occurred on other blockchain, i.e. 1 BTC was sent on Bitcoin Blockchain, and both sender and receiver have FIO Addresses. OBT stands for Other Blockchain Transaction

Parameters

fioRequestId - ID of funds request, if this Record Send transaction is in response to a previously received funds request. Send empty if no FIO Request ID

payerFioAddress - FIO Address of the payer. This address initiated payment.

payeeFioAddress - FIO Address of the payee. This address is receiving payment.

payerTokenPublicAddress - Public address on other blockchain of user sending funds.

payeeTokenPublicAddress - Public address on other blockchain of user receiving funds.

amount - Amount sent.

chainCode - Blockchain code for blockchain hosting this token.

tokenCode - Code of the token represented in Amount requested, i.e. BTC.

obtId - Other Blockchain Transaction ID (OBT ID), i.e Bitcoin transaction ID.

status - Status of this OBT. Allowed statuses are: sent_to_blockchain.

maxFee - Maximum amount of SUFs the user is willing to pay for fee. Should be preceded by /get_fee for correct value.

technologyPartnerId - FIO Address of the wallet which generates this transaction.

Exceptions

FIOError -

Return
PushTransactionResponse
