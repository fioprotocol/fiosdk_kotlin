requestFunds

fun requestFunds(payerFioAddress: String, payeeFioAddress: String, payeeTokenPublicAddress: String, amount: Double, chainCode: String, tokenCode: String, maxFee: BigInteger, technologyPartnerId: String = ""): PushTransactionResponse

Create a new funds request on the FIO chain.

Parameters

payerFioAddress - FIO Address of the payer. This address will receive the request and will initiate payment.

payeeFioAddress - FIO Address of the payee. This address is sending the request and will receive payment.

payeeTokenPublicAddress - Payee's public address where they want funds sent.

amount - Amount requested.

chainCode - Blockchain code for blockchain hosting this token.

tokenCode - Code of the token represented in amount requested.

maxFee - Maximum amount of SUFs the user is willing to pay for fee. Should be preceded by getFee for correct value.

technologyPartnerId - (optional) FIO Address of the wallet which generates this transaction.

Exceptions

FIOError -

Return
PushTransactionResponse
