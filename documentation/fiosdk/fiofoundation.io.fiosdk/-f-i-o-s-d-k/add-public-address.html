addPublicAddress

fun addPublicAddress(fioAddress: String, chainCode: String, tokenCode: String, tokenPublicAddress: String, maxFee: BigInteger, technologyPartnerId: String = ""): PushTransactionResponse

Adds a public address of the specific blockchain type to the FIO Address.

Parameters

fioAddress - FIO Address to add the public address to.

chainCode - Blockchain code for blockchain hosting this token.

tokenCode - Token code to be used with that public address.

tokenPublicAddress - The public address to be added to the FIO Address for the specified token.

maxFee - Maximum amount of SUFs the user is willing to pay for fee. Should be preceded by getFee for correct value.

technologyPartnerId - (optional) FIO Address of the wallet which generates this transaction.

FIOError -

PushTransactionResponse
